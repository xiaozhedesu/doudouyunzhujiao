# 豆豆云助教

这是我这学期开的微信小程序上的任务代码，同时也是我认为本学期最麻烦的课：内容比较多，而老师没有讲如何封装重复逻辑的方法，给出的示例代码也是一个函数用到底，request调用直接嵌套，js代码和wxml代码可读性很低。我不喜欢这种代码，所以我会尽量利用封装提高代码的可读性。既然改代码的苦都吃了，多少能体现我在前端的码风，我打算从今天（2025年11月4日）开始进行一下版本管理，做个记录。

## 遇到过的问题

- 当进行字段修改操作时，后端返回“修改成功”，但是重新启动模拟器时获得的数据并未被修改。由于我不知道它的后端具体做了什么，也没法问后端，询问老师也没有办法，只能作罢。
